# MSF

## Symbols

- ff - Free fall
- t - Throw
- sr - Slope release
- r - Resistance
- ar - Air resistance
- vt - Terminal velocity
- n\* - No + [any symbol]

The order of the symbols should be first the way the body behaves(i.e. free fall, throw, etc) then resistances and lastly valuable information that is given (i.e. terminal velocity).
